# Pharmacy Tracking System 🚀

A modern, robust, and intelligent Pharmacy Inventory Management System designed to optimize operations, track medication lifecycles, and provide automated demand forecasting.

## 🌟 Key Features

- **📊 Intelligent Dashboard**: Real-time analytics on stock levels, daily sales, and critical alerts.
- **📈 Demand Forecasting**: Built-in algorithm that predicts future demand based on historical sales data to prevent stockouts.
- **⏰ Expiration Tracking**: Automated monitoring of medicine expiration dates with proactive 30-day alerts.
- **📦 Inventory Management**: Complete lifecycle tracking of medications, including stock ownership (Main/Private) and manufacturer details.
- **💰 Smart Billing**: Integrated sales logging that feeds into the forecasting engine while providing instant customer billing.
- **🏢 Entity Management**: Centralized management for pharmaceutical companies and authorized agents.

## 🛠️ Technical Excellence

- **Core**: Java Swing with a modernized centralized architecture.
- **Database**: Centralized `DatabaseHelper` using Apache Derby (Network/Embedded hybrid).
- **Automation**: Self-initializing database schema and automated sales data modeling.
- **Portability**: Zero-configuration setup with local dependency management.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/utachicodes/PharmacyTrackingSystem.git
   ```
2. **Open in your favorite IDE**: (IntelliJ IDEA or NetBeans recommended).
3. **Run the Application**: The main entry point is `pharmacyinventorymanagement.PharmacyInventoryManagement`.
